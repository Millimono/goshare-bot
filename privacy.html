Politique de Confidentialité

Dernière mise à jour : 28 mai 2026

1. Introduction

GoShare Conakry ("nous", "notre") exploite un service de transport à la demande via WhatsApp. Cette politique explique comment nous collectons, utilisons et protégeons vos données personnelles.

2. Données collectées

Nous collectons les données suivantes :

  • Numéro de téléphone WhatsApp
  • Trajets demandés (point de départ et destination)
  • Historique des réservations
  • Informations sur les chauffeurs (trajet, heure, lieu, prix)

3. Utilisation des données

Vos données sont utilisées pour :

  • Mettre en relation passagers et chauffeurs
  • Gérer les réservations et confirmations
  • Améliorer la qualité du service
  • Communiquer avec vous via WhatsApp

4. Partage des données

Nous ne vendons pas vos données personnelles. Vos informations sont partagées uniquement avec :

  • Les chauffeurs pour confirmer les réservations
  • Les passagers pour faciliter la mise en relation
  • WhatsApp Business API (Meta) pour la messagerie

5. Conservation des données

Vos données sont conservées dans Google Sheets sécurisés. Vous pouvez demander la suppression de vos données à tout moment en nous contactant.

6. Sécurité

Nous utilisons des mesures de sécurité appropriées pour protéger vos données contre tout accès non autorisé.

7. Vos droits

Vous avez le droit de :

  • Accéder à vos données personnelles
  • Demander la correction de vos données
  • Demander la suppression de vos données
  • Retirer votre consentement à tout moment
